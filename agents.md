# Agent Instructions

## Operational Constraints

- **No Local Execution:** None of the scripts provided in this repository will be run locally on the development machine.
- **VM Testing:** All scripts and configurations must be tested manually on a Virtual Machine (VM) to ensure correct behavior in the target environment.
- **Target Environment:** The kit is designed for the AI Olympics (Olimpiada AI) and targets Windows machines with WSL 2 and Docker Desktop.

## Development Workflow

1.  **Code/Script Modifications:** Make necessary changes to the installation or post-installation scripts.
2.  **VM Preparation:** Ensure a clean VM environment is available for testing.
3.  **Manual Verification:** Transfer the kit to the VM and run the installation process manually, following the instructions in `README.md`.
4.  **Issue Reporting:** Document any failures or deviations observed during manual testing.
